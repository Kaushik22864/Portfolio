# Kaushik Raj Joshi — Portfolio

This repository contains a React single-page portfolio (Vite + React).

Quick start:

1. Install dependencies

```bash
npm install
```

2. Run dev server

```bash
npm run dev
```

3. Build for production

```bash
npm run build
```

Files:
- `index.html` — Vite entry that mounts the React app
- `src/` — React source files
	- `main.jsx` — app entry
	- `App.jsx` — main component and sections
	- `index.css` — styles
